# 临时邮箱技能 (Temp Mail)

## 描述
当用户需要邮箱验证码时，自动创建临时邮箱并检查收到的邮件。

## 使用方法

### 创建临时邮箱
```
创建一个新的临时邮箱
```

### 检查邮件
```
检查新邮件
```

### 获取验证码
```
查看最新邮件中的验证码
```

## 可用命令

| 命令 | 说明 |
|------|------|
| `temp-mail create` | 创建新的临时邮箱 |
| `temp-mail check` | 检查所有新邮件 |
| `temp-mail latest` | 获取最新邮件内容和验证码 |
| `temp-mail status` | 查看当前邮箱状态 |

## 工作流程

1. 当需要邮箱时，自动创建临时邮箱
2. 将临时邮箱地址提供给用户
3. 等待用户输入验证码或自动检查邮件
4. 从邮件中提取验证码/链接
5. 将结果返回给用户

## 注意事项

- 临时邮箱有效期有限，重要账户不建议使用
- 验证码通常以 4-8 位数字形式出现
- 也会提取邮件中的确认链接
